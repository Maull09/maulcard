AirBnB Learning Project

This is a description of my project which involved analyzing AirBnB data using Tableau. The project focused on three key areas: the average price per bedroom, price per zip code, and annual revenue, with the objective of gaining a better understanding of AirBnB pricing and revenue trends in Seattle during 2016.

To create easy-to-understand visualizations, I utilized AirBnB data from Seattle in 2016 and Tableau. The first step involved calculating the average price per bedroom and creating a bar chart to demonstrate how the average price varies depending on the number of bedrooms available. By visualizing this data, I was able to provide insights into the price range that can be expected based on the number of bedrooms available.

Secondly, I visualized the price per zip code using a map and also presented the data in a bar chart format. This enabled me to illustrate how prices differ across various areas in Seattle. This approach provides users with a better understanding of how Airbnb prices vary depending on location.

Lastly, I created a yearly income visualization to understand income trends during 2016. This visualization helps to comprehend how AirBnB revenue in Seattle changed during the course of the year.

Overall, the project helped me to improve my data processing and analysis skills using Tableau. I combined data analysis with informative and intuitive visualizations to provide a better understanding of Airbnb pricing and revenue trends in Seattle in 2016.

If you would like to see the full project, please follow this link: https://public.tableau.com/app/profile/mohamad.maulana.firdaus.ramadhan/viz/AirBnBLearningProject/Dashboard1
